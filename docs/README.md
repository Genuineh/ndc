## 快速开始

```bash
# 构建项目
cargo build --release

# 运行 CLI
./target/release/ndc --help

# 启动 REPL
./target/release/ndc repl

# 启用 Agent 模式 (REPL中)
ndc> /agent on

# 运行测试
cargo test --release
```
